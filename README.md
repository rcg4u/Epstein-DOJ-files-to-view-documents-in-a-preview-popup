# Epstein Files DOJ view helper

Chrome/Edge MV3 extension that:
- Auto-clicks the DOJ age verification **Yes** button (when present)
- Detects CAPTCHA presence (does not bypass)
- Provides an in-page hover preview window for PDFs, with pin/move/resize

## Load unpacked
1. Open `edge://extensions` or `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `extension` folder in this repo
